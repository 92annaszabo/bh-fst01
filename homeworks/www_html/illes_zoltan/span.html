A "<span>" -elem

a szövegszerkesztésben egyike a legkedveltebb elemnek.

Horgonyként használják a szövegre, vagy a szöveg egy részére.

Azt a szövegrészt amelyik a nyitó és záró tegjai között helyeszkedik el, kedvünk szerint lehetséges módosítani különböző atribútumokkal, CSS kódokkal, és még utólag is javascript funkciókkal.

Ez a szövegrész különböző módon lett módosítva:
beépített atribútumokkal,
a dokumentumban elhelyezett CSS kóddal,
sőt még az alábbi gombra kattintva egy funkció futtatásával is!
